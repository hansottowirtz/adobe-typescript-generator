colorAlphaAmountproperty 
colorAlphaAmount:int

Part of the color transformation for the instance. The same as using the Color > Advanced setting in the Instance PI and adjusting the controls on the right of the dialog. This either reduces or increases the tint and alpha values by a constant amount. This value is added to the current value. Most useful if used in conjunctin with colorAlphaPercent() Values allowed -255 to 255.
